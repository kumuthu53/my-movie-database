# Installation

**MyMovieDatabase** can be installed with either <code>pipx</code> or <code>pip</code>.

!!! warning
    **MyMovieDatabase** was built for Linux. It may not work properly on other platforms.

## Installing With <code>pipx</code> (Recommended)

```bash
pipx install my-movie-dataase
```

## Installing With <code>pip</code>

!!! warning
    It is always recommended to use a virtual environment when installing Python packages with <code>pip</code>. 

```bash
pip install my-movie-database
```
